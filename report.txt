INTRODUCTION

Knowledge distillation (KD) is a model compression technique that transfers 
learned representations from a large "teacher" model to a smaller "student" 
model. This report investigates how much of a ViT-Large teacher's 
representational power can be recovered in a ViT-Tiny student using diverse 
distillation strategies.

EXPERIMENT SUMMARY

Exp 1: Baseline (No Teacher)
  - Epochs: 100
  - Peak Val Top-1 Accuracy: 35.83%
  - Peak Val Top-5 Accuracy: 98.92%
  - Converged Val Loss: 0.6332 (Severe Overfit)

  Exp 2: ViTKD
  - Epochs: 20
  - Peak Val Top-1 Accuracy: 38.09%
  - Peak Val Top-5 Accuracy: 98.92%
  - Converged Val Loss: 0.5220

  Exp 3: CLS Token Distillation
  - Epochs: 20
  - Peak Val Top-1 Accuracy: 38.45%
  - Peak Val Top-5 Accuracy: 99.18%
  - Converged Val Loss: 0.3826


Exp 4: Attention Transfer Distillation
  - Epochs: 20
  - Peak Val Top-1 Accuracy: 34.92%
  - Peak Val Top-5 Accuracy: 98.28%
  - Converged Val Loss: 0.6478


DETAILED BREAKDOWN

  Experiment 1: Baseline Train (No Teacher)
    Configuration: Standard ViT-Tiny trained with Cross-Entropy Loss.
    Epochs: 100 | Learning Rate: 1e-4 | Optimizer: AdamW

  Experiment 2: ViT Patch Feature Distillation (ViTKD)
    Configuration: MSE loss between the projected student patch embeddings and 
    teacher patch embeddings, combined with Cross-Entropy classification loss.
    Epochs: 20 | Learning Rate: 1e-4 | Optimizer: AdamW

  Experiment 3: CLS Token Feature Distillation
    Configuration: Focuses feature distillation exclusively on aligning the CLS 
    token (representation index 0) between the student and teacher using a linear 
    projection layer.
    Epochs: 20 | Learning Rate: 1e-4 | Optimizer: AdamW
  

  Experiment 4: Attention Transfer Distillation
    Configuration: L2-normalized attention maps extracted over the spatial patch 
    dimensions. Uses MSE loss to transfer attention maps from teacher to student, 
    plus classification loss.
    Epochs: 20 | Learning Rate: 1e-4 | Optimizer: Adam
